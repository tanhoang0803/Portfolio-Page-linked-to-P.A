## Sections

| Section | Description |
|---------|-------------|
| **Hero** | Name, title, and call-to-action buttons |
| **Stats** | Projects count, years of experience, tools used |
| **Featured Projects** | Sales Analytics Dashboard + placeholder project cards |
| **Skills** | Analytics & Data · Frontend & Viz · Tools & Workflow |
| **Contact** | Email and GitHub links |

---

## Run Locally

```bash
npx serve . --listen 3000
# → http://localhost:3000
```

---

## Tech Stack

- **HTML5** — single-file, no build step
- **Tailwind CSS** (CDN) — utility-first styling with custom theme colors
- **Google Fonts** — Inter typeface

---

## Deploy

Auto-deploys via GitHub Actions on push to `main`.
Workflow: `.github/workflows/deploy.yml` (static upload, no build required).
GitHub Pages source must be set to **"GitHub Actions"** in repo Settings → Pages.
